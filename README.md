# Basil
For those with taste.

## Features
* Discover pure veg recipes with the ingredients available right now with AI.
* Explore the perfect veg recipe. Search for it and get the exact ingredients and way to make it with AI.
* Explore health recipes.
* Find pure veg restaurants near you.
* Create your own custom lists.
* Favorite recipes and browse through them for quick access.
* Check our journal for scientific reasons to choose vegetarian food.
* Set reminders to prepare and eat on time.
